# Pantheon Wars - Icon Specifications

**Total Icons Needed:** 88 (8 deity symbols + 80 perk icons)

**Format Requirements:**
- **Format:** PNG with transparency
- **Size:** 32x32 pixels (standard Vintage Story icon size)
- **Color Depth:** 32-bit RGBA
- **Style:** Pixel art or low-res fantasy style (consistent with Vintage Story aesthetic)

---

## Deity Symbols (8 icons)

These represent the main deities and will be used in religion selection, HUD displays, and religion browsers.

### 1. Khoras (War)
**File:** `assets/pantheonwars/textures/icons/deities/khoras.png`
**Description:** God of War, Battle, and Conquest
**Visual Concept:**
- Central Element: Crossed swords or battle axe
- Secondary Elements: Shield, flames, or blood drops
- Color Palette: Deep reds, oranges, metallic grays
- Style: Bold, aggressive, sharp edges
- Mood: Powerful, intimidating, martial

## Notes for Icon Generation

### Style Guidelines
1. **Consistency:** All icons should share a similar art style (pixel art or low-res fantasy)
2. **Clarity:** At 32x32, icons must be immediately recognizable
3. **Color Coding:** Each deity should have distinctive color themes
4. **Progression:** Tier progression should be visually apparent (simple → complex → godlike)
5. **Religion vs Player:** Religion perks should show group/multiple elements vs individual

### Naming Convention
- Use lowercase with underscores
- Match the perk ID structure
- No spaces or special characters

### Technical Requirements
- Transparent backgrounds (alpha channel)
- No anti-aliasing artifacts on edges
- Center important visual elements
- Test visibility at small sizes

### Tools Recommended
- **Pixel Art:** Aseprite, GraphicsGale, Piskel
- **AI Generation:** Midjourney, DALL-E, Stable Diffusion (with pixel art prompts)
- **Batch Processing:** ImageMagick for resizing/optimization
- **Optimization:** TinyPNG, pngquant for file size
