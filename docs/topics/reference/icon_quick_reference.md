## File Naming Convention

- All lowercase
- Use underscores for spaces
- Extension: `.png`
- Example: `warriors_resolve.png`

## Technical Specs

- **Size:** 32x32 pixels (exactly)
- **Format:** PNG with alpha channel
- **Background:** Fully transparent
- **File size:** Under 5KB (ideally under 2KB)
- **Color depth:** 32-bit RGBA

## Status Tracking

Use `/docs/icon_manifest.json` to track which icons are complete.
Update the `"status"` field for each icon as you generate them:
- `"pending"` - Not started
- `"in_progress"` - Currently working on
- `"complete"` - Finished and ready
- `"needs_revision"` - Generated but needs fixes

## Generation Order Recommendation

1. **Start with deity symbols** (8 icons) - Easiest, most important
2. **One deity at a time** (10 icons each) - Maintains consistency
3. **Player perks first** (6), then religion perks (4) - Natural progression

**Suggested order by difficulty:**
1. Khoras (War) - Simple combat imagery
2. Lysa (Hunt) - Nature/animal themes
3. Tharos (Storms) - Elemental effects
4. Aethra (Light) - Divine/holy themes
5. Gaia (Earth) - Nature/stone themes
6. Umbros (Shadows) - Dark/stealth themes
7. Morthen (Death) - Skull/undead themes
8. Vex (Madness) - Abstract/chaotic (hardest)
